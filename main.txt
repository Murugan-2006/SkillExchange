 SKILL EXCHANGE LEARNING PLATFORM (MERN STACK)

1️ SYSTEM OVERVIEW
The Skill Exchange Learning Platform is a web-based application where:
- Admins/Instructors create and manage courses
- Students enroll, learn, interact, submit projects, and earn certificates
- Learning is verified, interactive, and reward-based

Unlike traditional platforms, this system focuses on:
- Project-based certification
- Learning credits economy
- AI-powered quizzes
- Active learning engagement

2️ USER ROLES

 Student
- Register & login
- Browse and enroll in courses
- Watch course videos
- Participate in discussions
- Submit projects
- Earn certificates
- Earn & spend learning credits

 Admin / Instructor
- Login securely
- Create and upload courses
- Upload videos & materials
- Review student projects
- Approve/reject certifications
- Monitor student progress
- Manage learning credits

3️ STUDENT LOGIN – DETAILED FUNCTIONALITY

 Student Authentication
- Register with email & password
- JWT-based secure login
- Role-based access control

 Course Browsing & Enrollment
- View available courses
- Course details include:
  * Description
  * Skills covered
  * Credits required
- Enroll by:
  * Free enrollment OR
  * Spending learning credits
- Enrollment stored in database with progress tracking

🎥 Course Learning Flow
- Watch course videos
- Track:
  * Watch percentage
  * Last watched timestamp
  * Optional attention-aware playback (video pauses when attention is lost; student must opt-in for privacy)





 Project Submission
- After course completion:
  * Student submits:
    - GitHub repository link
    - Project description
  * Status:
    - Pending review
    - Approved
    - Rejected (with feedback)

🎓 
 Learning Credits System (Student)
- Students earn credits by:
  * Teaching peers
  * Answering questions
  * Reviewing projects (if permitted)
- Credits can be used to:
  * Unlock premium courses
  * Access advanced learning paths

4️⃣ ADMIN / INSTRUCTOR LOGIN – DETAILED FUNCTIONALITY

🔐 Admin Authentication
- Secure login
- Restricted admin routes
- Role-based authorization

📦 Course Management
- Admin can:
  * Create new courses
  * Define:
    - Course title
    - Description
    - Skills covered
    - Credits required
  * Upload:
    - Videos
    - PDFs
    - Assignments

🎥 Content Upload & Management
- Upload videos via:
  * Cloud storage
- Manage:
  * Video order
  * Transcripts (used for AI quizzes)

🧠 
🧪 Project Review & Approval
- Admin dashboard shows:
  * Submitted projects
  * Student details
  * GitHub links
- Admin actions:
  * Approve project → certificate issued
  * Reject project → feedback provided


💰 Learning Credit Control
- Admin:
  * Defines credit rules
  * Monitors abuse or fraud
 

5️⃣ DATABASE DESIGN (HIGH LEVEL)

Core Collections:
- Users (students, admins)
- Courses
- Enrollments
- Videos
- Discussions
- Credits
- Projects

6️⃣ SYSTEM ARCHITECTURE (MERN)

Frontend (React):
- Student dashboard
- Admin dashboard
- Video player
- Discussion UI

Backend (Node + Express):
- REST APIs
- Authentication
- Business logic


Database (MongoDB):
- Stores all users, courses, credits

7️⃣ SECURITY & PRIVACY

- JWT authentication
- Role-based access control
- HTTPS

Bonus Credits = Base × (Average Rating Score) × (Effective Volume Factor)
